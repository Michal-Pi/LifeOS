# Writing Agents Guide

## Thought Leadership Writer Preset

The Thought Leadership Writer preset is a graph workflow that builds content from strategy to
final polish.

## Workflow Stages

1. Strategy
2. Research
3. Draft
4. Edit + SEO (parallel)
5. Fact check
6. Final review

## Content Type Customization

Select a content type (e.g., blog post, case study) when instantiating the preset. Supported agent
templates will adjust prompts automatically for that content type.

## Example Prompts

- "Draft a thought leadership post about AI governance for SaaS executives."
- "Write a case study summary for a product-led growth initiative."
- "Create a newsletter-style explainer on data privacy changes in 2026."

## Tips

- Provide target audience, desired length, and tone.
- Include key points or sources you want emphasized.
- Review the fact check output before finalizing.
