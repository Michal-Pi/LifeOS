# Project Manager User Guide

## Overview

Project Manager adds structured questioning, assumption validation, and conflict detection to
workspaces. It helps clarify scope and align on decisions before execution.

## Enable and Configure

- Open a workspace and enable Project Manager in its configuration.
- Choose questioning depth (minimal, standard, thorough).
- Toggle conflict detection and assumption validation based on need.

## Using Clarification Questions

- Answer questions in the Project Manager panel to refine scope.
- Prioritize questions about timelines, constraints, and deliverables.
- Use Expert Council when questions require broader perspective.

## Resolving Conflicts

- Review detected conflicts and supply resolution notes.
- Select options that align with scope and feasibility.

## Questioning Depth

- **Minimal**: Quick clarification for simple tasks.
- **Standard**: Balanced approach for most workspaces.
- **Thorough**: Best for high-risk or complex planning.

## User Profiling

Project Manager tracks interaction patterns to infer expertise level:

- **Beginner**: More questions and guidance.
- **Intermediate**: Moderate guidance.
- **Expert**: Fewer questions and more autonomy.

Use these signals to adjust depth and workflow complexity.
